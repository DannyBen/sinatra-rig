My Sinatra Baseline
==================================================

A starting point for Sinatra apps.


In the Box
--------------------------------------------------

- [Runfile]
- Puma
- Slim
- SCSS
- CoffeeScript
- jQuery
- Rspec
- Vagrant compatibility
- Auto reloading
- Interactive IRB Console


Usage
--------------------------------------------------

- Clone the repository
- Run `bundle`
- Run `run server`
- Run `run --help` for more commands
